Detailed Description

Main program for BMP image processing.

Author
Maksim Pastukhov (st131.nosp@m.119@.nosp@m.stude.nosp@m.nt.s.nosp@m.pbu.r.nosp@m.u)

Provides a command-line interface for loading, manipulating, and saving BMP images. Supported operations include rotation and Gaussian blur filtering.
