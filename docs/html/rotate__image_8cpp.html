Rotates a BMP image by 90 degrees in the specified direction.

Specifies the direction of image rotation.

Parameters
imageThe BMPImage to rotate.
directionThe rotation direction (Clockwise or CounterClockwise).

The function:

  1. Creates a temporary buffer for rotated pixels.
  2. Splits the image into segments for parallel processing.
  3. Uses multiple threads to rotate each segment.
  4. Combines results and updates image dimensions.
